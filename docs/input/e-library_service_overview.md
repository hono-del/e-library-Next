# e-library サービス概要

## 1. サービス概要

**e-library** は、自動車メーカー向けに提供している情報閲覧サービスです。  
自動車メーカーが自動車販売店にアカウントを付与し、販売店の技術者や関連スタッフが各種マニュアルや技術情報を閲覧・共有できる仕組みです。

## 2. 利用状況

| 項目 | 内容 |
|---|---|
| サービス名 | e-library |
| 提供先 | 自動車メーカー |
| 主な利用者 | 自動車販売店 |
| 利用ディーラー数 | 約10ディーラー |
| 利用ユーザー数 | 約160ユーザー |
| 運用期間 | 10年以上継続 |

## 3. 主な機能

### ① ニュース

新しいマニュアルの追加や、システム更新などのお知らせを掲載する機能です。

主な掲載内容は以下の通りです。

- 新規マニュアル追加のお知らせ
- 既存マニュアル更新のお知らせ
- システム更新情報
- その他、メーカーから販売店への通知事項

---

### ② マニュアル

各種マニュアルを、車種・年式別に閲覧できる機能です。  
販売店の技術者やスタッフは、必要な車両情報を検索し、業務に活用できます。

閲覧可能な主なマニュアルは以下の通りです。

- サービスマニュアル
- オーナーズマニュアル
- 新車トレーニングマニュアル
- 車体修理マニュアル
- その他、車両整備・販売支援に関する資料

---

### ③ 情報交換

販売店やサービスショップの技術者同士が、修理事例や技術的な課題について情報共有する機能です。  
主に以下の2つのコンテンツで構成されています。

#### 1. TIEコンテンツ

TIEコンテンツとは、サービスショップの技術者やチームリーダーが作成する **TIEプロジェクト報告書** のことです。

この報告書には、車両修理中に遭遇した代表的な事例や困難な事例、およびその解決策が詳しく記載されています。

TIEコンテンツは、メーカーによる承認を経たうえで、TIEメニュー内の公式な共有レポートとして公開されます。

主な内容は以下の通りです。

- 車両修理中に発生した代表的な事例
- 対応が困難だった修理事例
- 問題の原因
- 対応プロセス
- 解決策
- 技術者・チームリーダーによる知見

#### 2. 問題交流

「問題交流」は、オンラインフォーラムのような機能です。

整備工場の技術者や関連スタッフが、車両修理中に直面したさまざまな問題について、他のオンラインスタッフに質問したり、回答や解決策を求めたりすることができます。

TIEコンテンツとは異なり、問題交流の投稿内容はメーカーの承認を必要としません。  
技術者同士の意見交換や、現場での課題解決を目的としたコミュニケーション機能です。

主な用途は以下の通りです。

- 修理中に発生した問題の相談
- 他の技術者への質問
- 解決策や対応方法の共有
- 現場知見の交換
- 類似事例の確認

## 5. サービスの特徴

e-libraryは、単なるマニュアル閲覧システムではなく、販売店・整備現場における情報活用基盤として機能しています。

主な特徴は以下の通りです。

- 自動車メーカーから販売店への情報共有が可能
- 車種・年式別に各種マニュアルを閲覧可能
- サービスマニュアルの利用頻度が高い
- TIEコンテンツにより、メーカー承認済みの修理事例を共有可能
- 問題交流により、現場技術者同士の知見共有が可能
- 10年以上継続運用されている実績がある
- 約10ディーラー・160ユーザーに利用されている

## 6. 位置づけ

e-libraryは、自動車販売店の整備・サービス業務を支援するための情報閲覧・技術共有プラットフォームです。

マニュアル閲覧に加えて、メーカー承認済みの修理事例や、技術者同士の問題交流を通じて、販売店現場における迅速な問題解決とナレッジ共有を支援しています。
